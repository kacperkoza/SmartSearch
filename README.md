# SmartSearch


Aplikacja na platformę Android stworzona w trakcie hakatonu BrainCode 2017. Pozwala wyszukiwać oferty w Allegro za pomocą głosu.
Dostępne parametry wyszukiwania:
- cena od x do y
- stan nowy / używany
- rodzaj licytacja / kup teraz
- miejscowość
- fraza wyszukiwania
